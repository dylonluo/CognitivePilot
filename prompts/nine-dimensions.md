# 九维框架提示词

深度分析框架，用于战略决策和高影响问题（25-30分钟）

## 分析步骤

### 1. 亚里士多德 - 第一性原理
- 这个问题的本质是什么？
- 哪些是事实，哪些是假设？
- 如果从零开始，我会怎么做？

### 2. 毛泽东 - 矛盾分析法
- 主要矛盾是什么？
- 矛盾的主要方面是什么？
- 解决主要矛盾后，次要矛盾会如何变化？

### 3. 麦肯锡 - 七步法+MECE
1. 定义问题
2. 分解问题（MECE原则）
3. 优先排序
4. 制定工作计划
5. 关键分析
6. 综合建议
7. 沟通展示

### 4. 丰田 - 5Why分析
- 连续追问5次"为什么"
- 找到根本原因
- 不要停留在表面症状

### 5. 黑格尔 - 正反合辩证法
- 正题：当前观点/方案
- 反题：对立观点/问题
- 合题：更高层次的综合方案

### 6. 德博诺 - 六顶思考帽
- 白帽：客观事实和数据
- 红帽：情感和直觉
- 黑帽：风险和问题
- 黄帽：优势和机会
- 绿帽：创新和可能性
- 蓝帽：流程和控制

### 7. 芒格 - 多元思维模型
从多个学科角度思考：
- 经济学：激励机制、机会成本、边际效应
- 心理学：认知偏差、社会认同、损失厌恶
- 生物学：进化论、适者生存、生态位
- 物理学：临界质量、杠杆原理、复利效应

### 8. 帕累托 - 二八法则
- 哪20%的因素产生80%的结果？
- 哪些是关键少数，哪些是次要多数？
- 如果只能做一件事，应该做什么？

### 9. 艾森豪威尔 - 优先级矩阵
```text
        紧急          不紧急
重要  | 立即做      | 计划做
不重要| 委托做      | 不做
```

## 适用场景

- 重大人生决策（辞职创业、职业转型等）
- 战略规划
- 复杂的商业决策
- 高影响的个人选择
- 需要深度思考的问题

## 注意事项

1. 九维分析需要25-30分钟，确保有足够时间
2. 不是所有问题都需要九维分析
3. 保持开放，每个维度都可能带来新的洞察
4. AI 提供分析，最终决策由用户做出
