# 灵活组合指南

## 核心理念

九维框架不是“必须用全部9个”的僵化模板，而是一个思维工具箱。根据问题性质、复杂度和时间限制，灵活选择和组合其中的几个维度。

## 组合原则

### 1. 问题导向

- 不是“我要用哪些维度”
- 而是“这个问题需要哪些维度”

### 2. 最小必要集

- 用最少的维度，解决最核心的问题
- 避免过度分析

### 3. 互补性

- 选择的维度应该互补，而不是重复
- 覆盖问题的不同方面

## 常见组合模式

### 模式1：快速决策（2-3个维度，3-5分钟）

适用：简单问题、时间紧迫、低风险

推荐组合：

- 第一性原理 + 二八法则
- 5Why + 艾森豪威尔矩阵
- 矛盾分析 + 正反合

### 模式2：标准分析（4-5个维度，10-15分钟）

适用：中等复杂度、需要全面思考、中等风险

推荐组合：

- 第一性原理 + MECE + 六顶思考帽 + 艾森豪威尔矩阵
- 矛盾分析 + 5Why + 正反合 + 二八法则
- 第一性原理 + 多元思维模型 + 六顶思考帽 + 艾森豪威尔矩阵

### 模式3：深度分析（6-9个维度，20-30分钟）

适用：高复杂度、重大决策、高风险

- 完整九维
- 或根据问题自定义组合

## 按问题类型选维度

### 技术问题

核心维度：

- 第一性原理
- 5Why
- 二八法则

可选维度：

- MECE
- 艾森豪威尔矩阵

### 人际问题

核心维度：

- 矛盾分析
- 六顶思考帽
- 正反合

可选维度：

- 第一性原理
- 多元思维模型

### 商业决策

核心维度：

- 第一性原理
- MECE
- 多元思维模型
- 二八法则

### 战略规划

核心维度：

- 第一性原理
- 矛盾分析
- MECE
- 多元思维模型
- 正反合
- 艾森豪威尔矩阵

### 个人成长

核心维度：

- 第一性原理
- 5Why
- 六顶思考帽
- 艾森豪威尔矩阵

## 自定义组合流程

1. 识别问题类型：技术 / 人际 / 商业 / 战略 / 个人成长
2. 选择核心维度：先定 2-3 个
3. 添加补充维度：再补 1-3 个
4. 验证组合：是否覆盖关键方面、是否冗余、是否超时
